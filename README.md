Запуск проекта:   
docker compose up -d - запустит проект на 8000 порту
в корневой папке должен былть .env без нее конфигурацию базы и редиса не поднимится
после нужно сделать миграцию: php artisan migrate 

Авторизация: 
Работает следующим образом: 
    Сначало нужно зарегестрироваться через метод register
    (Примечание: почта должна быть дейстительно и быть gmail)
    После получения OTP кода нужно отправить запрос на метод я verify c указанием почты и OTP
    Далее Логинизация для получения JWT токена закрытые апи по избражения без наличия JWT не работаю

В корневой папке есть postman_collection его нужно импортировать в postman и вся колекция высетится
